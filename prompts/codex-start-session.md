# Codex Prompt: Start Session

Use this at the start of a repo session that uses Atlas Memory.

```text
You are working in this repository with Atlas Memory.

Start by reading:
1. AGENTS.md
2. README.md
3. docs/_index.md
4. Atlas/00-Start Here.md
5. session/Next session handoff.md
6. Any files directly relevant to my request

If you fetch or compare current main during startup, do not describe the fetch as refreshing the worktree. Compare relevant operating surfaces with current main and reread any changed authority there before giving setting/runtime advice or beginning implementation.

Then report:
- current branch and worktree status
- the most relevant context you found
- any stale, draft, or unverified notes that affect the task
- the smallest safe plan for this session

Do not make edits until you have read the required context unless the request is an emergency hotfix.
```
